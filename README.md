# 🎓 SkillHub – Course Marketplace

SkillHub is a modern full-stack Course Marketplace where students can discover, enroll in, and manage online courses. The platform features secure authentication, role-based dashboards, course management, and a responsive user interface built with TypeScript.

# 📦 Installation

## Clone Repository

```bash
git clone <frontend-repository-url>
```

## Frontend

```bash
cd frontend

npm install

npm run dev
```

---

## Backend

```bash
cd backend

npm install

npm run dev
```

# ⚙ Environment Variables

## Frontend (.env.local)

```env
BETTER_AUTH_SECRET=Your_Better_Auth_Secrect
BETTER_AUTH_URL=Your_local_or_deployment_urls
MONGO_URI=Your_MongoDb_URI
NEXT_PUBLIC_SERVER_URL=Your_Backend_Server_urls
NEXT_PUBLIC_IMAGE_AP=ImgBB_api_urls
```

---

## Backend (.env)

```env
PORT=5000

MONGO_URI=YOUR_MONGODB_URI

```
